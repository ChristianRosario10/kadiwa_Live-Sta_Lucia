KAPISANAN LIVE System User Guide

About KAPISANAN LIVE

KAPISANAN LIVE is a comprehensive member management system designed specifically for Kadiwa organizations.

Dashboard Overview

The dashboard provides a quick overview of your organization's current status:

  • Total number of registered Kadiwa members
  • Count of members with assigned tungkulin
  • Status-based member counts (Aktibo, Di Aktibo, UWP, MS, May Sakit)
  • Recent member registrations in List A

Member Registration

To register a new member:

  1. Navigate to the "Register Member" page
  2. Fill in the required information:
    • Full Name
    • Birthday
    • Contact Number (11 digits)
    • Number of Tungkulin (0-5)
    • Member Status
  3. If adding tungkulin, enter the titles in the dynamic fields
  4. Click "Register Member" to submit

Note:

All fields marked with * are required. The system will automatically generate a unique Kadiwa Serial Number for each new member.

Admin Panel

The Admin Panel provides advanced management capabilities:

  • Edit member information
  • Update member status
  • Manage tungkulin assignments
  • Delete member records
  • View complete member history

Security Note:

Admin access is restricted to authorized personnel only. Please keep your login credentials secure.

System Limitations

Please be aware of the following limitations:

  • Maximum of 5 tungkulin per member
  • Contact numbers must be 11 digits
  • Data is stored locally in your browser
  • No automatic backup system (regular exports recommended)

Getting Help

If you encounter any issues or have questions:

  • Check this guide for relevant information
  • Contact your system administrator
  • Ensure you're using a modern web browser
  • Clear your browser cache if experiencing display issues
